## Project Description
The description of the project that was done is known to be a Weather Dashboard which is  built using JavaScript and the OpenWeatherMap API. this allows the users to search for any city by also viewing the current weather conditions,to include temperature, weather description, humidity, wind speed, pressure, and a 5-day weather forecast. The application also stores the five most recent searches using localStorage for quick access.

The project was completed as the Week 6 assignment to practice asynchronous JavaScript concepts such as Promises, Async/Await, the Fetch API, DOM manipulation, API integration, and localStorage.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- OpenWeatherMap API
- Local Storage

## Features
- Searching weather by city
- Displaying the wind speed
- Displaying of the atmospheric pressure
- A 5-day weather forecast
- By Loading indicator while fetching data
- Handling error for invalid city names and network errors
- Recent searches history
- Save recent searches using localStorage
- Click a recent search to load the weather again

## Lessons Learned
 I learned how to:

## Challenges Faced
The  challenges I encountered were just just but this includes;
- Implementing recent searches using localStorage.
- Displaying a 5-day forecast using the forecast API endpoint.
- Understanding how asynchronous functions work with Async/Await.
- Handling errors returned by the API, such as invalid city names.
- Dynamically updating multiple DOM elements with weather data.

These challenges were solved by carefully structuring the code into reusable functions, using try/catch blocks for error handling, and testing the application with different city names.
