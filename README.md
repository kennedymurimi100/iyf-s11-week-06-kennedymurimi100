# Weather Dashboard

## Description

A weather dashboard built with JavaScript using the OpenWeatherMap API.

## Features

- Search weather by city
- Current temperature
- Weather description
- Weather icon
- Feels like temperature
- Humidity
- Wind speed
- Pressure
- Loading state
- Error handling
- Recent searches
- Local storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- OpenWeatherMap API
